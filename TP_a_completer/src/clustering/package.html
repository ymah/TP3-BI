Contient toutes les classes et interfaces utiles pour l'algorithme de clustering, en particulier :

  • Donnee : n-uplet de réels (double), n étant appelé nombre de dimensions.
  • Cluster : ensemble de données, avec quelques statistiques sur ces données (par exemple la moyenne ou barycentre).
  • Clustering : algorithme de clustering, méthode des k-means.
  • Distance : interface qui décrit ce qu'est une distance entre deux données.
