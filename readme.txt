TP3 - Clustering

Utilisation :
- se placer dans le dossier TP_a_completer/bin
- executer la commande suivante
	java Main***** nbClusters choixDeLaDistance [option pour minkowski]
		pour choixDeLaDistance : 'euclidienne', 'manhattan' ou 'minkowski'
		pour minkowski un paramètre supplémentaire (un double) est nécessaire

exemeple : java exemples.MainSquare ../../square1.data 3 euclidienne

Le programme exécute l'algo de clustering, affiche les étapes et les nouveaux centres recalculés à chaque étapeet affiche à la fin BC, WC et le rapport BC/WC
